This app contains top 50 most starred repositories in the Github. To find the desired repository it has two types of sorting - 1. most popular  2. last updated

The app locally persist the data so that user can comfortably use the app even with no internet connection. All the functionalities will perfectly run without internet availability. The user can also see the details of any particular repo.

To ensure any future scopes, the app has been built robustly with scalability opportunities in future. 

The app will normally run and it has below notable packages used -

1. Bloc
2. Dartz
3. Injectable
4. GetIt
5. Dio
6. Equatable
7. Shared Preferences
